# cat_vasya_game
Ролевая система с пошаговой боевкой, реализованная на Python с соблюдением принципов ООП.

## Модули

### Модуль 1. Базовая архитектура
- Абстрактный класс `Unit` с 6 характеристиками.
- Классы `Character` и `Monster` с уникальными формулами здоровья, урона и защиты.
- Округление результатов до целого в меньшую сторону.

### Модуль 2. Классы персонажей
- Удалён `Monster`, оставлен только `Character`.
- Добавлен атрибут `character_class` (`warrior`, `mage`, `hunter`).
- Реализовано ветвление формул урона и защиты внутри одного класса.
- Валидация класса персонажа при создании.
- Предварительный расчёт `max_health`, `damage`, `defense` в конструкторе.

### Модуль 3. Магия и мана
- Абстрактный класс `Spell` и конкретные заклинания: `Fireball`, `IceLance`, `LightningBolt`.
- Добавлены `mana` и `spells` в `Unit`.
- Реализованы методы `add_spell()` и `cast_spell()` с проверкой маны.
- Формулы максимальной маны для каждого класса.
- Автоматическое заполнение маны при создании персонажа.
